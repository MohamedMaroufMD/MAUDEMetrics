About MAUDEMetrics
FDA Medical Device Adverse Event Explorer
Supporting Evidence-Based Medical Device Safety Research
MAUDEMetrics is an open-source tool for exploring adverse event reports submitted to the FDA's MAUDE database, enabling efficient safety signal detection and reporting.

Project Overview

MAUDEMetrics provides researchers, healthcare professionals, and medical device safety analysts with powerful tools to efficiently search, filter, and export comprehensive adverse event data for evidence-based safety research and signal detection.

The platform integrates directly with the FDA's MAUDE (Manufacturer and User Facility Device Experience) database through the openFDA API, ensuring access to the most current and comprehensive medical device safety data available.

Key Features

Advanced Search

Search by brand names, product codes, manufacturers, product class, and date ranges with flexible partial matching and case-insensitive search capabilities.

Analytics Dashboard

Comprehensive analytics with demographic summaries, event type distributions, manufacturer analysis, and interactive charts with modern multi-select filters.

Advanced Data Export

Professional data exports with optimized sheets: Events (with integrated MDR texts), Summary, and fast Raw Data export in original API structure (ZIP with 5 CSV files).

Safety Focus

Designed specifically for medical device safety research and signal detection in healthcare settings with comprehensive data validation.

Search Capabilities

Search Fields
  • Brand Name(s) - Partial matching
  • Product Code(s) - FDA codes
  • Manufacturer(s) - Company names
  • Product Class(es) - Generic names
  • Date Range - Start/End dates
Search Features
  • Case-insensitive search
  • Partial word matching
  • Multiple values per field
  • Flexible date ranges
  • Cumulative search results

Data Source

MAUDEMetrics utilizes the FDA's openFDA API to access the MAUDE (Manufacturer and User Facility Device Experience) database, which contains reports of adverse events involving medical devices.
