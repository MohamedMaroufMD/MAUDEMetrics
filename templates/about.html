About MAUDEMetrics
FDA Medical Device Adverse Event Explorer
Supporting Evidence-Based Medical Device Safety Research
MAUDEMetrics is an open-source, no-code desktop application and web service for automated extraction, standardization, and export of adverse event reports from the FDA's MAUDE database — enabling reproducible, large-scale device safety research without programming expertise.

Project Overview

MAUDEMetrics provides researchers, healthcare professionals, and medical device safety analysts with powerful tools to efficiently search, filter, and export comprehensive adverse event data for evidence-based safety research and signal detection.

The platform integrates directly with the FDA's MAUDE (Manufacturer and User Facility Device Experience) database through the openFDA API, ensuring access to the most current and comprehensive medical device safety data available.

Key Features

Advanced Search

Search by brand names, product codes, manufacturers, product class, and date ranges with flexible partial matching and case-insensitive search capabilities.

Analytics Dashboard

Comprehensive analytics with demographic summaries, event type distributions, manufacturer analysis, and interactive charts with modern multi-select filters.

Advanced Data Export

Professional data exports with flexible options: Optimized Events (with integrated MDR texts), standalone Summary Statistics, and fast Raw Data export in original API structure (ZIP with 5 CSV files).

Safety Focus

Designed specifically for medical device safety research and signal detection in healthcare settings with comprehensive data validation.

Search Capabilities

Search Fields
  • Brand Name(s) - Partial matching
  • Product Code(s) - FDA codes
  • Manufacturer(s) - Company names
  • Product Class(es) - Generic names
  • Date Range - Start/End dates
  • FDA API Key (recommended for 2x faster downloads)
Search Features
  • Case-insensitive search
  • Partial word matching
  • Multiple values per field
  • Flexible date ranges
  • Advanced pagination (search_after) for massive datasets

Technical Details

Backend & Data
  • Python 3.11+ / Flask 2.3.3
  • search_after cursor pagination — no record ceiling
  • SQLite local cache — repeat queries in <1 s
  • Pandas + NumPy standardization pipeline
  • OpenPyXL memory-safe streaming export
  • Exponential-backoff retry & rate-limit handling
Desktop App & Deployment
  • Electron 28 — signed macOS & Windows installers
  • PyInstaller — standalone Python bundle
  • Docker + Docker Compose — containerized deployment
  • Auto-update via GitHub Releases
  • Bootstrap 5 — responsive UI & interactive charts
  • Real-time extraction console (Server-Sent Events)

Data Source

MAUDEMetrics utilizes the FDA's openFDA API to access the MAUDE (Manufacturer and User Facility Device Experience) database, which contains reports of adverse events involving medical devices.

Powered by openFDA: This application uses the openFDA API but is not endorsed by or affiliated with the FDA.
Public Data & Legal Compliance: All data is retrieved exclusively through the official openFDA API — the FDA's own platform for programmatic public access. MAUDE data is released by the FDA under the Creative Commons CC0 1.0 Universal dedication (public domain). No proprietary or restricted endpoints are accessed.
